IBM Japan Constructs Large-Scale Web System for Online Game Service

December 17, 1999 (TOKYO) -- IBM Japan Ltd. announced details of what it claims is one of the world's largest Web systems for Randnet, an online game service operated by RandnetDD Co., Ltd.

RandnetDD is a joint venture of Nintendo Co., Ltd. and Recruit Co., Ltd. The online game system uses a combination of Nintendo's game machine, NINTENDO64, and its 64DD disk drive as a client. The system can be used to download and upgrade game software, view Web sites as well as transmit and receive e-mail.

Starting this month, RandnetDD started providing the first phase of services. The online system is "one of the world's largest commercial Web systems," IBM Japan's company spokesman said.

With the "RS/6000 SP," IBM Japan's large-scale parallel server with 24 processor chips (four CPUs x six nodes), as its core hardware, the large Web system consists of about 30 "RS/6000" units, the firm's UNIX machine for authentication and security management, three "AS/400" units for membership management and six units of "Netfinity," a PC server for content management.

The company realized a Java-based run-time environment "Web Sphere" for Web applications on the online game system. The company also adopted "DB2 Universal Database" as the database and "Network Dispatcher" as the load distribution tool. These tools are improved technologies over those the company had introduced for Nagano Winter Olympics held in 1998. Therefore, according to the spokesman, the system can tolerate simultaneous access from 100,000 users and can process its jobs with no trouble.

IBM Japan also manages the entire system and provides outsourcing services for maintenance and management of the system.
